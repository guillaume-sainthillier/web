Déploiement

Pour déployer l'application, veuillez suivre ces indications:

$ cd path/to/www
$ git clone https://github.com/guillaume-sainthillier/iscore.git

Installation

Aucune configuration n'est requise, il suffit juste d'un serveur web pour accéder au contenu!

Pré requis

Vous devez avoir installé:

  • un serveur apache et php
  • une base de données : mysql ou postgresql

En fait, reportez vous aux installations de :

  • sous windows : wamp (http://www.wampserver.com/) ou easyphp(http://easyphp.fr/)
  • sous linux : lamp

Si vous utilisez postgre sql, voir install sur le site : http://www.postgresqlfr.org/

Si vous debutez, il est plus simple de garder mysql qui est packagee avec easyphp ou wamp.

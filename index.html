TP Web IUT Blagnac

Les TP Web de l'I.U.T de Blagnac ont pour but de se familiariser avec le XHTML combiné avec la mise en forme CSS, et aborde également les notions de PHP et Javascript

Auteur

Guillaume Sainthillier guillaume.sainthillier@gmail.com

Récupération

Si vous possédez Git, tapez dans un invité de commande:

$ cd path/to/www
$ git clone https://github.com/guillaume-sainthillier/tp-web-blagnac.git

Si vous ne possédez pas Git (ce qui est déconseillé), téléchargez le projet au format Zip ou Tar.gz, et extrayez le dans le dossier www de votre serveur apache.

Déploiement

Aucune configuration n'est requise, il suffit juste d'un serveur web pour déployer le site!

Pré requis

Vous devez avoir installé:

  • un serveur apache

En fait, reportez vous aux installations de :

  • sous Windows : WAMP ou easyphp
  • sous Linux : LAMP

Notez que le serveur apache est uniquement nécessaire pour le TP9 partie PHP
